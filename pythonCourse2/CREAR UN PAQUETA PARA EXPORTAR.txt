1) CREAMOS UN ARCHIVO "__init__.py" DENTRO DEL PAQUETE PARA EXPORTAR O RECIEN
CREADO PARA ASÍ ESTE PUEDA SER UTILIZADO.

2) LUEGO CREAMOS EN EL DIRECTORIO DE LA CARPETA CONTENEDORA UN ARCHIVO "setup.py"
DONDE CONTENDREMOS LOS DATOS QUE SE UTILIZARÁN PARA EXORTACIÓN QUE DEBE TENER LOS
SIGUIENTES DATOS:

from setuptools import setup

setup(
   name="packagePoo",
   version="1.0",
   description="Package of POO",
   author="Emir Zahir",
   author_email="emir@hotmail.com",
   url="www.linkedin.com",
   packages=["poo"]

)

3) POR ÚLTIMO COLOCAMOS EN EL SIMBOLO DEL SISTEMA EL SIGUIENTE COMANDO PARA EJECUTARLO

python setup.py sdist

4) NOS CREA UNA CARPETA "dist" CON UN ARCHIVO COMPRIMIDO DENTRO LISTA PARA EXPORTAR
O ENVIAR POR EMAIL, ETC.
 
5) AL MOMENTO DE RECIBIRLO SE DEBE UTILIZAR EL SIGUIENTE COMANDO EN EL SIMBOLO DE
SISTEMA PARA PODER UTILIZARLO QUE ES:

  cd dist
  pip install "EL NOMBRE COMPLETO DEL ARCHIVO COMPRIMIDO CON SU EXTENSIÓN"

6) AHORA SE PODRA UTILIZAR DESDE CUALQUIER PARTE DEL EQUIPO YA QUE FUE INSTALADO
EN NUESTRO SISTEMA. Y PARA DESISTALARLO SE DEBE USAR EL SIGUIENTE COMANDO.

  pip unistall "NOMBRE DEL PAQUETE"
