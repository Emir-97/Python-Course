HAY VARIAS FORMAS PARA REALIZARLO:
 1)LA PRIMERA CREA TRES CARPETAS, LA DE NOMBRE DIST Y BUSCAR EL ARCHIVO .EXE SI
SE HIZO EN WINDOWS Y EJECUTARLO. ESTE SERÍA EL MÁS LIVIANO PERO EL QUE TRAE UNA
VENTANA DE SITEMA Y QUEDA CON FEA ESTETICA

  pyinstaller  calculator.py

2) LA SEGUNDA NO TIENE LA VENTANA DEL SIMBOLO DEL SISTEMA, PEROCREA UN ARCHIVO 
 PESADO PERO DE FACÍL EJECUCIÓN Y EXPORTACIÓN
 
    pyinstaller --windowed --onfile calculator.py

3) LA ÚLTIMA SE PUEDE MOFICAR EL ICONO DEL ARCHIVO PERO DEBE SER DE UNA EXTENCIÓN
".ico" PARA LA IMAGEN QUE SE UTILICE COMO LOGO DEL ARCHIVO EJECUTABLE. HAÍ SE VE
UN EJEMPLO DE LA MISMA

 pyinstaller --windowed --onefile --icon=./icon.ico calculator.py
